2020再战浪潮ipbs9505

之前有写过一篇<浪潮ipbs9505 刷机破解研究>,当时因为没有TTL转USB工具无法通过软件的方法对浪潮ipbs9505进行刷机破击,这次在有了TTL转USB工具工具下简直就是分分钟钟的事情.

准备

  1. TTL转USB版(某宝入手11.1元-土豪金 CH340G-USB转TTL-USB转TTL USB转串口下载线CH340G模块 RS232升级板刷机板线PL2303 )
  2. 公对母杜邦线(3.32-杜邦线 母对母 公对母 公对公40P彩色排线连接线10/15/30/20/40CM )
  3. U盘一个
  4. 自备电烙铁和拆机设备

破解

将TTL连(焊)到盒子上对应的位置

注意按照GND-GND RX-TX TX-RX 连接!!!!

版的背面有写,可以参考下图

下载盒子桌面软件,如沙发桌面或者当贝桌面,将其下载到U盘,并连接到盒子

接下来准备使用putty连接(驱动按TTL转USB商家提供的教程安装)

1.点击Serial

2.COM端口号同设备管理中的COM端口号相同

3.点击Open

4.盒子通电

通电之后，会开始跑代码，跑完代码后就变成可以输入的状态
root@xxxxxx:/ #

接着开始”破解”安装第三方软件

错误

Error: Activity not started, unable to resolve Intent { act=android.intent.action.CALL }

盒子桌面 精简去广告版 使用 am start com.dangbei.tvlauncher命令会出现类似错误,且不能正常启动,解决方案暂无.只能使用官方版本,推荐沙发桌面广告少点

用户您好,本终端已关闭第三方应用的直接安装,请从移动应用商场安装,谢谢!

解决方法:安装re管理器,通过复制的方法将/storage/external_storage/sda1U盘里面的apk复制到/data/app也可以一定程度的避免软件的乱下载或误点广告或软件升级问题.
