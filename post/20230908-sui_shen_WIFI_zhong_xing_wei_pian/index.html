随身WIFI中兴微的一些简单操作

无需刷web即可开启功能后台

1:首先先开ADB 用水遍助手 https://www.123pan.com/s/8y49-oBZ0hUSB 连接电脑。

  • http://192.168.0.1/goform/goform_set_cmd_process?goformId=ALK_EXC_AT_CMD1&AT_CMD1=ZCARDSWITCH%3D0
  • http://192.168.0.1/goform/goform_set_cmd_process?goformId=SET_DEVICE_MODE&debug_enable=2
  • http://192.168.0.1/goform/goform_set_cmd_process?goformId=SET_DEVICE_MODE&debug_enable=1
  • http://192.168.0.1/reqproc/proc_post?goformId=SET_DEVICE_MODE&debug_enable=1

2:开启完后网页进 高级后台 修改IMEI
登录密码是：factoryAdmin
进高级设置/其他/AT 命令

AT+MODIMEI=你要改的串号-改完后点恢复出厂设置

如果没有AT 命令也可以

进高级设置/其他/修改IMEI-改完后点恢复出厂设置，效果是一样的

注 : 之所以先改串不先切卡是因为改完串要恢复出厂设置，恢复完后切卡就是默认esim的了。

恢复出厂设置后WIFI名称会变成【厂商指定名称＿你指定的串号后６位】

3:接着admin进入后台 高级功能 SIM卡解锁

密码可能是一下几个之一：

  • az952#
    • 疆训
  • IMEI 号后 6 位 + 999
    • TZ103_V1.1版本号

解锁后会自动重启然后操作切卡即可。

切卡

4:打开 水遍助手 去后门
按 F） ZXIC去远控(v3.0) 再按 2.传统zxic通过adb 去后门。去完后会重启。

去后门

5:重启后再次打开助手
按 T). CMD终端
依次输入以下内容:

删除远控脚本
rm: can’t remove ‘/sbin/tc_tbf.sh’: Read-only file system

频段选择：
http://192.168.0.1/index.html#band_selection
家长控制：
http://192.168.0.1/index.html#child_ctl
副SSID：
http://192.168.0.1/index.html#wlan_guset
VPN：
http://192.168.0.1/index.html#vpn
TR069：
http://192.168.0.1/index.html#tr069
DDNS：
http://192.168.0.1/index.html#dynamic_dns

现在就可以进后台看到 功能后台了
(注一条一条的输 入)
