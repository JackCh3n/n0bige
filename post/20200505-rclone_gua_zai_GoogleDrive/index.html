rclone挂载GoogleDrive

Posted on 2020-05-05 In 技巧 Views:

适用于

  • Windows[Linux]系统下使用rclone挂载GoogleDrive

特点

  • 网络运行的情况下，将GoogleDrive当硬盘使用。

准备

  • rclone ： https://rclone.org/downloads/
  • winfsp (rclone依赖库)： http://www.secfs.net/winfsp/download/
  • Google Api：第一次需要启用然后再点击创建

开始

Windows下：rclone和winfsp根据自己系统版本选择。下载winfsp 依赖库后正常安装即可，rclone 则解压到某个文件夹即可

Linux下：安装可以参考linux常用和错误集

创建配置文件过程通用，可以参考下面配置，需要手动操作的地方搜索 #

总需要输入api ID和 api token,目录文件夹，两个N即可
