Apache常用命令

查看apache版本 启动apache 重启apache 关闭apache
sudo apachectl -v sudo apachectl start sudo apachectl restart sudo apachectl stop

Apche常用的几个文件和目录

主目录 /etc/apache2/
默认日志目录 /private/var/log/apache2
日志1 /private/var/log/apache2/mysites-error_log.log
日志2 /private/var/log/apache2/mysites-access_log.log
主配置文件 /etc/apache2/httpd.conf
虚拟主机文件 /etc/apache2/extra/httpd-vhosts.conf
